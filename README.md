# mbnf-parser
A basic BNF parser following [Feller, David, et al. "What Does This Notation Mean Anyway?." (2018)](https://arxiv.org/abs/1806.08771)


#### Run example with:
`stack build && stack exec mbnf-parser-exe example/Grammar.mbnf`
